# To-Do List em PHP com MySQL

Este é um projeto simples de **To-Do List (Lista de Tarefas)** desenvolvido em **PHP** com **MySQL**, utilizando **PDO** para conexão com o banco de dados.  
O objetivo do projeto é praticar **CRUD**, organização de código e integração com banco de dados, partindo de uma ideia simples e funcional.

---

## Funcionalidades

- Criar novas tarefas
- Listar tarefas cadastradas
- Editar tarefas existentes
- Excluir tarefas
- Marcar tarefas como **concluídas**
- Separação visual entre tarefas pendentes e concluídas
- Armazenamento em banco de dados MySQL
- Interface simples e objetiva

---

## Tecnologias Utilizadas

- **PHP**
- **MySQL**
- **PDO**
- **HTML5**
- **CSS3**
